RiftLink

Web Flash Tool

Прошивка устройств через браузер с выбором модели и автоматической загрузкой правильного manifest.

Перед прошивкой

  • Используйте Chrome/Edge/Opera (нужен Web Serial).
  • Подключайте USB-кабель с поддержкой передачи данных.
  • Для V4 при необходимости войдите в bootloader через PRG.
